## Proffy
 
The Proffy platform was made at NLW,
and aims to facilitate the meeting between
teachers and students.
 
 
## Technology 
 
Here are the technologies used in this project.
 
* Javascript
* Node.Js 12.18.3
* Express 4.17.1
* Nunjucks 3.2.2
* Nodemon 2.0.4
* Sqlite-Async 1.1.0
* HTML5
* CSS3
